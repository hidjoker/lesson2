
 + this
  - 자기객체 참조

  - 인스턴스가 자기 자신을 참조할 때 사용

  - 메소드 내부에서 메소드의 지역변수와 멤버필드가 같은 이름일 때
   멤버필드(인스턴스변수)를 지칭하기 위해 사용됨

	ex)
		private int num;

		public void setNum(int num) {
			this.num = num;
			//멤버필드 = 지역변수;
		}


 + this()
  - this생성자

  - 자기 객체의 다른 생성자 호출

  - 생성자가 오버로딩되어 있는 상황에서
   객체가 생성될 때 다른 생성자를 재사용하기위해 사용하는 코드

  ** 코드재사용 : 만들어져있는 코드를 다른 곳에서 다시 사용하는 것


  - 생성자에서 가장 처음으로 호출해야만 한다
