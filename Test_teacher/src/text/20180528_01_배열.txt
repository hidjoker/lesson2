
 + 배열, Array
  - 같은 타입의 여러 개 변수를 하나의 묶음으로 사용하는 것

  - 여러개의 변수를 한번에 선언 가능

  - 선언되는 여러 개의 변수의 이름을 고민할 필요가 없다



 + 배열 선언

  데이터타입[] 배열명; //추천

  데이터타입 배열명[];

	ex) int[] arr;


 + 배열 생성

  new 데이터타입[갯수];



 ex)
	int[] arr;
	arr = new int[100];


 + 배열의 접근
  - 배열의 인덱스(index)를 이용하여 배열공간에 접근

  - 인덱스는 숫자 0 부터 시작

  - 음수는 올 수 없다

  ex)
	arr[0] = 10;
	arr[10] = 1234;
	arr[100] = 555;	// 만약 new int[100]으로 생성하면 접근불가
