
 + 자료구조, Data Structure

  - 데이터를 효율적으로 저장하는 전략

  - 효율성

  - 특정 목적(알고리즘)에 맞는 저장 구조

  - 탐색 능력(성능)

  - 저장 능력(공간)



 + 컬렉션, Collection
  (컬렉션 프레임워크)

  - 자바에서 제공하는 자료구조

  - List, Set, Map




 + 배열, Array

  - 자료구조의 일종

  - 가장 기본적인 형태의 자료구조

  - 같은 데이터타입만 관리한다

  - 인덱스 사용 (0 시작)
