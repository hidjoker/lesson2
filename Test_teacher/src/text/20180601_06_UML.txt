
 + UML, Unified Modeling Language
  - 기획, 설계, 구현으로 이루어지는 프로그램 개발 절차에서
   프로젝트참가인원(기획자, 디자이너, 개발자, 사장 등등)들이
   의사소통을 원활하게 하기 위해 사용되는 표준화된 모델링 언어

  - 스토리보드, Story Board
	화면구현을 위한 설계

  - 유스케이스 다이어그램, Use-case Diagram
	요구사항을 기반으로 프로그램의 기능적 요소들을 표현한 것

	프로그램을 동작시키는 행위자를 중심으로 표현

  - 클래스다이어그램, Class Diagram
	프로그램의 클래스를 표현한 것 (정적)

  - 시퀀스다이어그램, Sequence Diagram
	클래스의 동작을 표현한 것 (동적)

  - 패키지다이어그램, Package Diagram
	클래스의 패키지들을 표현한 것
