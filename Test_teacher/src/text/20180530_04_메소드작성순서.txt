
 + 메소드, method
  - 함수, function

  - 클래스의 기능(동작)을 구현한 코드

  - 클래스의 객체를 통해서 호출해야 한다

  - 객체 간의 데이터전달(커뮤니케이션) 수단이 된다


 + 메소드 정의 방법

	[접근제한자] [리턴타입] [메소드명] ( 매개변수 ) {

		// 실행코드

		// 구현부(메소드의 동작)
	}


 + 메소드 작성 순서

  0. 메소드의 기본형태 작성

	public void method() {
	}

  1. 메소드 이름 정하기 -> 만들려는 기능을 대표하는 이름으로

  2. 매개변수 작성 -> 기능을 수행하는데 필요한 변수

  3. 코드 작성 -> 수행할 기능을 구현

  4. return 코드 작성 -> 기능완료에 대한 출력 데이터

  5. return Type 작성 -> return 코드에 맞는 데이터 타입
