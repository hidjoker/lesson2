
 + 객체지향 프로그래밍
  - Object Oriented Programming, OOP

  - 객체들의 상호작용을 통해 프로그램이 동작되도록
   프로그래밍하는 기법


  - 객체(Object) : 프로그램으로 구현하고자하는
		  현실세계의 모든 대상

  - 클래스(Class) : 객체를 추상화하여 표현한 코드


  ** 추상화 : 객체에 공통으로 존재하는, 속성, 기능, 행위들을
	개념화 시키는 작업

	-> 프로그램에서 필요한 요소들만 뽑아서 정리하는 작업

	-> 주요 관심대상만 고려한다

	-> 불필요하거나 중요하지 않은 대상은 무시
