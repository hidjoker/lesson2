
 + 프로그램 오류, Error

  1. 컴파일 타임 에러, Compile-time Error

	컴파일 시 발생하는 에러
	( 이클립스에서는 소스를 저장했을 때 )

	주로 문법 에러, Syntax Error


  2. 런타임 에러, Run-time Error

	프로그램 실행 도중에 발생하는 에러

	수행할 수 없는 작업을 시도할 때 발생

	ArrayIndexOutOfBoundsException

	NullPointerException

	ArithmeticException


  3. 논리 오류, Logical Error

	 실행은 정상적으로 이루어지지만
	프로그램의 흐름이 개발자의 의도대로 동작하지 않을 때

	ex)
		무한루프

		데이터 오버플로우
